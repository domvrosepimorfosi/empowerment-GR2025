Κείμενο Πλαισίωσης
«Οι γονείς/κηδεμόνες είναι συνεργάτες/ιδες. Η σαφήνεια υπερτερεί της “βαριάς” ορολογίας· το τρακ σημαίνει ότι νοιάζεσαι· η παρουσία μετρά περισσότερο από την ένταση. Στόχευσε στη σύνδεση αντί για την “παράσταση”.»
Γιατί σε αφορά:
  • Η τακτική, στοχευμένη επικοινωνία ενισχύει τη δέσμευση και τη συνέπεια στο πλάνο δράσης.
  • Η σχολική-κοινή εμπλοκή οικογενειών πολλαπλασιάζει το αποτύπωμα.
  • Η ψύχραιμη αποκλιμάκωση + σαφή όρια προλαμβάνουν την κλιμάκωση.
  • Τήρησε τον GDPR: χρησιμοποίησε εγκεκριμένα κανάλια· προστάτευσε τα δεδομένα του/της μαθητή/τριας.
Δραστηριότητα Γ1 — Αποδόμηση Μύθων (Parent Edition)
Στόχος
Αντικαθιστάς μη βοηθητικούς μύθους με πρακτικές συνήθειες.

Μύθος → Πρακτική αντικατάσταση

  • «Πρέπει να απομνημονεύεις κάθε λέξη.» → Χρησιμοποίησε δομή/ιστορία· μίλησε σε άνθρωπο, όχι σε «σενάριο».
  • «Οι “μεγάλες” λέξεις σε κάνουν να φαίνεσαι ειδικός/ή.» → Η απλή γλώσσα χτίζει εμπιστοσύνη· μετάφρασε την ορολογία.
  • «Το τρακ σημαίνει ότι είσαι κακός/ή ομιλητής/τρια.» → Τρακ = νοιάζεσαι· ονόμασέ το, ανάπνευσε, συνέχισε.
  • «Πρέπει να μιλάς δυνατά.» → Ρυθμός–παύση–έμφαση καλύτερα από την ένταση.
Δραστηριότητα Γ2 — Τηλεφωνική Επικοινωνία με Γονέα/Κηδεμόνα
BIOS = Boundary • Intent • Observation • Step

Ενδεικτικές ελληνικές φράσεις (συμπλήρωσε [μαθητή/τρια])

  • Boundary/Χρόνος: «Καλησπέρα σας. Έχω περίπου 4 λεπτά για να μοιραστώ κάτι χρήσιμο για το παιδί σας.»
  • Intent/Πρόθεση: «Σας καλώ γιατί θέλω ο/η [μαθητής/τρια] να αισθάνεται ότι τα καταφέρνει στα Αγγλικά και να στηρίξουμε μαζί την πρόοδό του/της.»
  • Observation/Παρατήρηση (χωρίς κρίση): «Σήμερα παρατήρησα ότι ξεκίνησε, αλλά σταμάτησε μετά από τρεις προτάσεις.»
  • Step/Επόμενο βήμα: «Δύο ιδέες που βοήθησαν στην τάξη ήταν μια αρχή πρότασης και ένα δίλεπτο διάλειμμα. Ποια θα λειτουργήσει στο σπίτι; Θα σας στείλω απόψε ένα μικρό υπόδειγμα.»
Δραστηριότητα Γ3 — Email Templates που Ρίχνουν τους Τόνους

Template A — Έννοια/Ανησυχία (ουδέτερο)

Θέμα: Σύντομη ενημέρωση για τον/την [Μαθητή/Μαθήτρια] — [Μάθημα]

Καλησπέρα [Όνομα], παρατηρώ [συγκεκριμένη συμπεριφορά/αποτέλεσμα] κατά τη διάρκεια [πλαισίου]. Στο σχολείο βοηθά το [στρατηγική]. Μπορούμε να δοκιμάσουμε στο σπίτι το [μία μικρή ενέργεια] αυτή την εβδομάδα; Θα επανέλθω [ημερομηνία] για να δούμε πώς πήγε. Ευχαριστώ για τη συνεργασία.

Template B — Θετική «Μικρο-Νίκη»

Καλησπέρα [Όνομα], μικρή νίκη σήμερα: ο/η [Μαθητής/Μαθήτρια] [συγκεκριμένη επιτυχία]. Αύριο θα το αξιοποιήσουμε με [επόμενο βήμα].

Template C — Επιβεβαίωση Συνάντησης (GDPR-safe)

Καλησπέρα [Όνομα], επιβεβαιώνω ότι θα συναντηθούμε [ημερομηνία/ώρα] για το θέμα [θέμα]. Θα συζητήσουμε μόνο τη μάθηση του/της [μαθητή/τριας] (όχι άλλους/ες μαθητές/τριες). Μετά τη συνάντηση θα στείλω σύντομη σύνοψη ενεργειών. Παρακαλώ αποφύγετε την αποστολή ευαίσθητων στοιχείων με email· χαίρομαι να τα συζητήσουμε από κοντά.

Δραστηριότητα Γ4 — Playbook Συνάντησης με Γονέα/Κηδεμόνα

Διαχείριση

  • Έναρξη: «Σας ευχαριστώ που ήρθατε. Στόχος μας σήμερα είναι να συμφωνήσουμε σε ένα πλάνο που θα βοηθήσει τον/την [μαθητή/τρια] να προοδεύει στα Αγγλικά.»
  • Ακρόαση: «Θα ήθελα να ακούσω την κυριότερη ανησυχία σας σε μία πρόταση.» → «Αν καταλαβαίνω σωστά, …»
  • Μοίρασμα δεδομένων: σύντομα δεδομένα & δείγματα· χωρίς «ταμπέλες».
  • Συν-σχεδιασμός: «Έχω δύο επιλογές να δοκιμάσουμε· ποια ταιριάζει καλύτερα;»

Κλείσιμο

  • «Συμφωνούμε να δοκιμάσουμε: [ενέργειες].»
  • «Θα στείλω σύνοψη μίας παραγράφου και θα επικοινωνήσω [ημερομηνία].»
  • «Προτιμάτε email ή τηλέφωνο; Ευχαριστώ για τη συνεργασία.»
Δραστηριότητα Γ5 — Ατάκες Ορίων

Χρήσιμες ατάκες

  • Acknowledge + Pivot: • «Σας ακούω. Ας το βάλουμε στο πλάνο για να το βελτιώσουμε.»
  • Κανόνας χωρίς ένταση + Προσφορά: • «Πρέπει να τηρώ την πολιτική του σχολείου, οπότε μπορώ να κάνω σήμερα το εξής…»
  • Stop the shout: • «Θέλω να σας ακούσω. Δεν μπορώ όσο οι φωνές είναι ψηλά. Θα περιμένω.»
  • Επανεστίαση:” • «Ας μείνουμε στη δουλειά του/της [μαθητή/τριας]. Δεν μπορούμε να συζητήσουμε για άλλους/ες μαθητές/τριες.»
  • Όριο χρόνου: • «Έχω 10 λεπτά τώρα· αν χρειαστούμε περισσότερο, θα κανονίσουμε μεγαλύτερο ραντεβού.»
  • Ασφαλής κλιμάκωση: • «Ας σταματήσουμε εδώ και ας επανέλθουμε με τον/την [Διευθυντή/Υποδιευθυντή] για να το λύσουμε σωστά.»
Δραστηριότητα Γ6 — Τύποι Γονέων/Κηδεμόνων & Κινήσεις

1) Θυμωμένος/η (δυνατά, γρήγορα)

Κινήσεις: Χαμήλωσε/επιβράδυνε τη φωνή· αντανάκλαση σε 1 πρόταση → δύο επιλογές.

Ατάκα: «Είστε αναστατωμένος/η επειδή θέλετε ο/η [μαθητής/τρια] να πετύχει. Έχω δύο επιλογές — να διαλέξουμε μία;»

2) Αγχωμένος/η (ανησυχεί, επαναλαμβάνει)

Κινήσεις: Κανονικοποίηση, σύνοψη, μικρο-πλάνο.

Ατάκα: «Πολλές οικογένειες νιώθουν έτσι. Ορίστε ένα μικρό πλάνο για αυτή την εβδομάδα· θα το ελέγξω την Παρασκευή.»

3) «Νομικός» Γονέας (τεκμήρια, πολιτική)

Κινήσεις: Μένεις σε γεγονότα, έγγραφα, πολιτική· χωρίς εικασίες.

Ατάκα: «Εδώ είναι τα σημερινά δείγματα και η ρουμπρίκα αξιολόγησης. Η πολιτική μας λέει [x]· μέσα σε αυτό, μπορούμε να προσφέρουμε [y].»

4) Υπερ-εμπλεκόμενος/η (micromanaging)

Κινήσεις: Διευκρίνισε ρόλους· όρισε «βοηθώ» vs «το κάνω εγώ».

Ατάκα: «Η καθοδήγησή σας είναι πολύτιμη. Για να χτίσουμε αυτονομία, ο/η [μαθητής/τρια] θα κάνει τα βήματα 1–2· εσείς μπορείτε να ελέγξετε το βήμα 3.»

5) Δύσκολη Επικοινωνία (δεν απαντά)

Κινήσεις: Εγκεκριμένα κανάλια, σύντομο μήνυμα + ένα σαφές αίτημα· προγραμμάτισε check-in.

Ατάκα: «Σύντομη ενημέρωση: [ένα γεγονός]. Μπορείτε να επιβεβαιώσετε [ναι/όχι]; Θα σας καλέσω στις 14:00 αν δεν έχω απάντηση.»

Δραστηριότητα Γ7 — Τεκμηρίωση & Μίνι-Λίστα Ελέγχου GDPR

Μονοσέλιδο αρχείο επικοινωνίας

Ημ/νία & Ώρα • Κανάλι • Γονέας/Κηδεμόνας • Σύνοψη (γεγονότα) • Συμφωνημένες ενέργειες • Ημερομηνία επανελέγχου • Θέση αποθήκευσης

Συνήθειες ασφαλείς για GDPR

  • Χρησιμοποίησε συστήματα που εγκρίνει το σχολείο· απόφυγε προσωπικά μηνύματα για μαθητικά θέματα.
  • Κοινoποίησε το ελάχιστο αναγκαίο· ποτέ για άλλους/ες μαθητές/τριες.
  • Επιβεβαίωσε ταυτότητα πριν τηλ. κοινοποίηση λεπτομερειών.
  • Προτίμησε σύντομες, πραγματολογικές σύνοψεις μετά τη συνάντηση έναντι μακρών threads.
Δραστηριότητα Γ8 — Επίδειξη Αποκλιμάκωσης
  1. Αναπνοή + στάση σώματος.
  2. Ονόμασε το συναίσθημα: «Ακούγεστε πραγματικά απογοητευμένος/η.»
  3. Μία ερώτηση: «Ποιο είναι το πιο σημαντικό που χρειάζεται να φτιάξουμε πρώτο;»
  4. Δύο επιλογές: «Μπορούμε να προσαρμόσουμε την εργασία για το σπίτι ή να ορίσουμε εβδομαδιαίο σύντομο ραντεβού ελέγχου.»
  5. Κλείσιμο με follow-up: «Θα σας στείλω το πλάνο με email· ξανασυναντιόμαστε την επόμενη Πέμπτη.»
Δραστηριότητα Γ9 — Κάρτες Ατζέντας & Έτοιμα Σενάρια

Ατζέντα

Στόχος • Ανάγκη/Ανησυχία Γονέα • Τεκμήρια • Επιλογές • Απόφαση • Follow-up

«Δύσκολες» καταστάσεις — Έτοιμες ατάκες

  • Φήμη από social media: «Δεν μπορώ να μιλήσω για άλλους/ες μαθητές/τριες ή ιδιωτικά ζητήματα. Αυτό που μπορώ να επιβεβαιώσω για τη δουλειά του/της [μαθητή/τριας] σήμερα είναι… Ας εστιάσουμε εκεί.»
  • Ηχογράφηση χωρίς συναίνεση: «Η ηχογράφηση/βιντεοσκόπηση συναντήσεων δεν επιτρέπεται εδώ. Θα σας στείλω αμέσως μετά μια σύντομη περίληψη με email, ώστε να υπάρχει επίσημο αρχείο.»
  • Υπόσχεση που δεν μπορείς να τηρήσεις: «Δεν υπόσχομαι κάτι που δεν μπορώ να τηρήσω. Ορίστε τι μπορώ να κάνω αυτή την εβδομάδα.»
  • Κλιμάκωση εκτός αρμοδιότητας: «Αυτό χρειάζεται τη συμβολή της διοίκησης. Θα προσκαλέσω τον/την [Διευθυντή/Υποδιευθυντή] ώστε να αποφασίσουμε μαζί.»
